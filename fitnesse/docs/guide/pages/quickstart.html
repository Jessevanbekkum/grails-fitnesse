2.1 Quickstart - Reference Documentation

Authors: Erik Pragt (jWorks.nl), Marcin Erdmann (jWorks.nl)

Version: 2.0.1

2.1 Quickstart

For the impatient, this is the quickstart to get Fitnesse to work with Grails.

Step 1: Install the Fitnesse Plugin

grails install-plugin fitnesse

Step 2: Start the Grails application

grails run-app

Step 3 (in a differtent console or terminal): Start Fitnesse

grails run-fitnesse

Step 4: Configure Fitnesse (for Linux/MacOS)

Browse to http://localhost:9090, and paste the following code (See 'Edit' in the left side menu) into the Fitnesse wiki: 

!define TEST_SYSTEM {slim}
!define COMMAND_PATTERN {echo}

The COMMAND_PATTERN needs to be something which executes, but does noting. On Windows for example this can an empty batch file.

Step 5: Add the Test

Paste the following also into the Fitnesse wiki: 
|import                         |
|nl.jworks.fitnesse.quickstart|
|echo|
|input|output?|
|hello|  hello|
|world|  world|

Save the page.

Step 6: Verify it works

Click on Properties, select Test for 'Page type', click on Save properties to save the page.

Now press Test. Everything should be Green now!
